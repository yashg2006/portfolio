# 🚀 Yash Gharat | Modern Portfolio

A high-performance, aesthetically driven professional portfolio featuring advanced scroll-synchronized video scrubbing, glassmorphism UI, and modular architecture.

**Live Demo:** [https://yashg2006.github.io/portfolio/](https://yashg2006.github.io/portfolio/)

---

## ✨ Key Features

- **Smooth Video Scrubbing**: High-fidelity background videos that sync perfectly with user scroll progress using optimized All-Intra (GOP 1) encoding.
- **Glassmorphism UI**: Modern, premium design aesthetic with frosted-glass effects and subtle micro-animations.
- **Dynamic Projects Section**: Interactive showcase of AI, IoT, and Cybersecurity projects with high-impact visual indicators.
- **Experience Timeline**: Structured overview of professional internships and research roles.
- **Performance Optimized**: Refactored logic using `requestAnimationFrame` and `IntersectionObserver` for buttery-smooth 60FPS performance.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Video Processing**: FFMPEG (optimized for web scrubbing)
- **Design**: Glassmorphism, Responsive Grid, Intersection Observer API
- **Fonts**: Space Mono, DM Sans

## 📁 Project Structure

```text
├── index.html       # Main entry point (Modular structure)
├── style.css        # Centralized design system & variables
├── script.js       # Refactored scroll-scrubbing & animation logic
├── video.mp4       # Optimized hero background (All-Intra)
└── video2.mp4      # Optimized projects background (All-Intra)
```

## 🚀 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yashg2006/portfolio.git
   ```
2. Open `index.html` in any modern web browser.
3. To achieve the smooth scrubbing effect locally, ensure the videos remain in the root directory.
